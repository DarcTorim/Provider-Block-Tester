 📋 Описание

**Provider Block Tester** — это утилита для проверки доступности веб-ресурсов и определения типа блокировки со стороны интернет-провайдера. Программа проводит многослойную диагностику (DNS, HTTP, TCP).

 🔍 Что умеет программа:

| Возможность | Описание |
|-------------|----------|
| **DNS-проверка** | Сравнение ответов системного и публичного DNS |
| **HTTP-тест** | Проверка доступности сайта по протоколу HTTPS |
| **TCP-сканер** | Проверка доступности портов (80, 443, 8080, 8443) |
| **Определение типа блокировки** | DNS, DPI, TCP Reset, TCP Block |
| **Рекомендации** | Автоматические советы по обходу блокировок |
